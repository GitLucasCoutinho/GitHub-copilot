DIO Cars - exemplo MVC com Flask

Como rodar (Windows PowerShell):

```powershell
python -m pip install -r requirements.txt
python app.py
```

Abra no navegador: http://127.0.0.1:5000/

Descrição: página que mostra uma tabela "DIO Cars" com 5 carros fixos. O botão "Selecionar" dispara uma requisição que exibe a mensagem: "voce selecionou o carro (NOME_DO_CARRO)".
